Tables

Usage notes

  1. Are tables flexible?

    Yes, they are. You can find the torrent of table variants.

  2. How to use them properly nowadays?

    The only rule: make them look as if they are in a book and read the getting started post by Adrian Roselli.

  3. Can tables be responsive?

    Sure. And the answer can be flexible. There are data best to be presented as a list, image, separate table. The Responsive Accessible Table article by Adrian Roselli helps us.

  4. What is the list of steps for me? And how can I use JavaScript?

    A Complete Guide to the Table Element by Chris Coyier covers all the table questions.
